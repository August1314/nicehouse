### 一句话概念

在 Unity 里，**“给空调加代码”= 给代表空调的 GameObject 挂一个 C# 脚本组件**，脚本里写逻辑，UI 或别的脚本通过调用它来开关/调节。

---

### 步骤示例：给“空调”添加控制脚本

以你们项目为例，建议这样做：

1. **在场景中找到空调物体**
   - 在 `Hierarchy` 里选中代表空调的那个 `GameObject`（可能叫 `AC` / `AirConditioner` / 某个 Mesh 的名字）。  
   - 如果模型是一大坨，可以在 `Hierarchy` 里展开，给真正的空调部分单独建一个父物体，比如 `AirConditioner`.

2. **创建脚本文件**
   - 在 `Project` 面板中（建议新建目录）：  
     - 右键 `Assets` → `Create` → `Folder` → 命名为 `Scripts`（再细一点可以 `Scripts/Devices`）。  
     - 在 `Scripts/Devices` 里右键 → `Create` → `C# Script` → 命名：`AirConditionerController`.
   - 双击脚本用 VS / VS Code 打开。

3. **写一个最简单的控制逻辑（示例）**

```csharp
using UnityEngine;

public class AirConditionerController : MonoBehaviour
{
    public bool isOn = false;           // 当前是否开启
    public float targetTemperature = 24f; // 目标温度

    // 这里可以引用灯光、风扇、UI 文本等
    public GameObject fanObject;

    public void TogglePower()
    {
        isOn = !isOn;
        UpdateState();
    }

    public void SetTargetTemperature(float temp)
    {
        targetTemperature = temp;
        // 根据需要做一些动画/显示更新
    }

    private void UpdateState()
    {
        // 例子：开关一个风扇模型
        if (fanObject != null)
        {
            fanObject.SetActive(isOn);
        }
        // 这里还可以加声音、材质变化等
    }
}
```

4. **把脚本挂到空调对象上**
   - 回到 Unity，选中 `Hierarchy` 里的空调 `GameObject`。  
   - 在 `Inspector` 里：  
     - 把 `AirConditionerController` 脚本拖到空调上，或点击 `Add Component` 搜索 `AirConditionerController` 添加。  
   - 如果脚本里有 `public` 字段（比如 `fanObject`），在 `Inspector` 里把对应的子物体拖进去绑定。

5. **让 UI/按钮去调用空调的函数**
   - 比如你有一个 UI 按钮用来“开关空调”：  
     - 选中按钮 → `Button (Script)` 组件 → `On Click ()` 事件列表中点击 `+`。  
     - 把场景里的空调对象拖到那个空格。  
     - 右边选择函数：`AirConditionerController -> TogglePower()`。  
   - 以后按钮一按，就会执行你脚本里的 `TogglePower()`，空调就能按你写的逻辑响应。
