## 系统架构草图（待补充）

> 建议由全组一起讨论后完善，本文件只给出初步框架。

### 1. Unity 场景结构（建议）

- `MainMenu`：主菜单场景（开始演示、选择模式）
- `HouseScene`：主三维好房子场景
  - 数字人
  - 房屋结构（客厅、卧室、厨房、卫生间等）
  - 传感器/设备可视化元素

### 2. 功能模块划分（示例）

- 环境智控模块
- 智能监护模块
- 健康监测模块
- 一键呼叫模块
- 安全防护模块
- 分层查看/数据可视化模块（温度、湿度、PM2.5、活动热力图等）

### 3. 技术点

- 摄像机控制（自由观察房屋、切换视角）
- UI 与交互（按钮、面板、提示信息）
- 动画与时间线（设备开关、数字人行为）
- 数据模拟与可视化（例如温度分布、热力图）
